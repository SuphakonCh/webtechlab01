รายละเอียดโปรเจกต์ (Project Description)
--------------------------------------------------
โปรเจกต์นี้เป็นเว็บไซต์ E-commerce สำหรับการขายผักและผลไม้ออร์แกนิก โดยพัฒนาขึ้นในส่วนของ Front-end (UI/UX) ประกอบไปด้วยหน้าหลักๆ ดังนี้:
1. หน้าแรก (Home) - แสดงสินค้าแนะนำ, โปรโมชั่น, และหมวดหมู่สินค้า
2. หน้าร้านค้า (Shop) - แสดงรายการสินค้าทั้งหมด สามารถค้นหาและกรองตามหมวดหมู่ได้
3. หน้าตะกร้าสินค้า (Cart) - สำหรับจัดการสินค้าที่ต้องการสั่งซื้อ
4. หน้าชำระเงิน (Checkout) - สำหรับกรอกข้อมูลการจัดส่งและชำระเงิน
5. หน้าติดต่อเรา (Contact) - สำหรับช่องทางการติดต่อและแผนที่ร้าน

เทคโนโลยีที่ใช้ (Technologies Used)
--------------------------------------------------
- HTML5
- CSS3
- Bootstrap 5 (Framework สำหรับจัด Layout และ Responsive Design)
- JavaScript / jQuery
- FontAwesome & Bootstrap Icons

วิธีการใช้งาน (How to run)
--------------------------------------------------
1. ดาวน์โหลดหรือ Clone โปรเจกต์นี้ลงในเครื่อง
2. เปิดไฟล์ `index.html` ผ่านเว็บบราวเซอร์ (เช่น Google Chrome, Firefox, Safari) เพื่อดูผลลัพธ์
